Use esta URL como referência para configurar integrações externas e webhooks.

Esta é a URL base oficial da API para integrações externas
💡 Código para ativar ngrok: ngrok http 3000

Endpoints e Eventos:

POST /api/appointments - Criar agendamento
appointment_created - Quando um novo agendamento é criado
PUT /api/appointments/:id - Atualizar agendamento
appointment_updated - Quando um agendamento é atualizado
DELETE /api/appointments/:id - Excluir agendamento
appointment_deleted - Quando um agendamento é excluído
GET /api/slots/:date - Horários disponíveis
slots_retrieved - Quando os horários disponíveis são consultados
💡 Dica: Use estes endpoints para configurar webhooks e outras integrações externas.
