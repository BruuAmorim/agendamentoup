📋 Endpoints Principais:

GET /api/slots/:date - Horários disponíveis
POST /api/appointments - Criar agendamento
PUT /api/appointments/:id - Atualizar agendamento
DELETE /api/appointments/:id - Excluir agendamento
GET /api/health - Status da API
💡 Dica: Use estes endpoints para configurar webhooks no n8n e outras integrações externas.

🔗 Integração n8n

Configure o webhook do n8n para receber notificações automáticas sobre eventos de agendamentos.

URL completa do webhook criado no n8n para receber eventos
Quando ativada, o sistema enviará eventos para o webhook configurado

📋 Eventos Enviados

O sistema envia automaticamente os seguintes eventos para o webhook:

  • appointment_created - Quando um novo agendamento é criado
  • appointment_updated - Quando um agendamento é atualizado
  • appointment_cancelled - Quando um agendamento é cancelado
  • appointment_deleted - Quando um agendamento é excluído
